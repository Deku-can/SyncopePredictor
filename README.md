# SyncopePredictor v1.2

Програма для моніторингу пацієнтів під час фізичної реабілітації та прогнозування вазовагального синкопе (непритомності). Розроблено як дипломний проєкт.

##  Як запустити готову програму (Для комісії)
1. Переконайтеся, що на комп'ютері встановлено **Java (JDK 17 або новіше)**.
2. Завантажте файл `SyncopePredictor.jar` з розділу **Releases** (праворуч на сторінці вашого GitHub).
3. Двічі клацніть на завантажений файл для миттєвого запуску програми.

*Альтернативний запуск через термінал або командний рядок:*
```bash
java -jar SyncopePredictor.jar
```

##  Скомпілювати з вихідного коду (Для розробників)
Якщо ви хочете самостійно зібрати проєкт безпосередньо з файлу `SyncopePredictor.java`:
```bash
javac SyncopePredictor.java
java SyncopePredictor
```

##  Функціонал та логіка роботи
* **Імітація датчика Garmin**: Симуляція надходження R-R інтервалів у реальному часі через вбудований таймер.
* **Прогнозування синкопе**: Автоматичне виявлення падіння децелераційної здатності серця (DC) нижче критичного медичного порогу 4.5 мс.
* **Експорт звітів**: Збереження графіку змін та показників ЧСС у формат `.CSV` для подальшого аналізу лікарем.
